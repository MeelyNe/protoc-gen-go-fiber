# protoc-gen-go-fiber

`protoc-gen-go-fiber` 是一个用于生成基于 [Go](https://golang.org/) 和 [Fiber](https://gofiber.io/) 的 Web 服务的 Protocol Buffers (protobuf) 文件的插件。

## 安装

使用以下命令安装 `protoc-gen-go-fiber`：

```shell
go install github.com/your-username/protoc-gen-go-fiber
```
